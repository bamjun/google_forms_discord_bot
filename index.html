구글 폼 디스코드 봇 사용 설명서

1. 디스코드 봇 추가

먼저 디스코드 서버에 봇을 추가해야 합니다. 봇 초대 링크를 사용하여 원하는 서버에 봇을 초대하세요.

2. 구글 폼 URL 및 입력창 ID 찾기

구글 폼의 URL과 입력창 ID를 찾아야 합니다. 방법은 다음과 같습니다:

  1. 구글 폼에 접속합니다.
  2. 브라우저의 개발자 도구를 엽니다 (보통 F12 키).
  3. 네트워크 탭을 선택합니다.
  4. 설문 조사를 실행합니다.
  5. 'formResponse' 항목을 찾습니다.
  6. 페이로드에서 'entry.' 뒤에 있는 숫자를 찾습니다. 이것이 입력창 ID입니다.

3. 봇 설정하기

디스코드 채팅에서 다음 명령어를 사용하여 봇을 설정합니다:

/setting [구글폼URL] [입력창ID]

예시:

/setting https://docs.google.com/forms/d/e/your-form-id/viewform 1234567

4. 메시지 보내기

설정이 완료되면, 다음 명령어를 사용하여 구글 폼에 메시지를 보낼 수 있습니다:

/add [메시지]

예시:

/add 안녕하세요, 이것은 테스트 메시지입니다.
